le variabili in JavaScript:

var

la variabile var è stata la variabile principale di js fino a prima dell'aggiunta di let e const(2015),
questa può essere:
una stringa: es. x = "marco";
console.log(x);// output:marco
un numero: es.var x = 10;
var y = 2 * x;
console.log(y);//output 20
un array: cos'è un array?
un array è un insieme di stringhe,numeri o oggetti in una sola variabile,
es. var x =[1,2,3,4,5,6,7,"marco","giuseppe","antonio"];
console.log(x);//output 1,2,3,4,5,6,7,marco,giuseppe,antonio
un oggetto: cos'è un oggetto?
un oggetto in js è un insieme di dati chiamati property ovvero propietà le quali sono personalizzabili
es.var x ={
nome:"marco",
cognome:"rossi",
anni:18
};
console.log(x);//output nome:marco,cognome:rossi,anni:18

let

let è attualmente la variabile più usata in js ed è praticamente la versione migliore di var.
let se viene dichiarata due volte da errore a differenza di var(altro motivo per cui non si usa più).
anche questa variabile può essere una stringa,un numero,un array o un oggetto come var.

const

const a differenza di var e let è una constante cioè una volta assegnatogli un valore questo non cambia,
anche questa come var e let può essere una stringa,un numero,un array o un oggetto.

come distringuere le variabili

per capire di che tipo è una variabile ci sono delle semplici differenze:
stringhe:le stringhe sono dichiarate con "" o '' o ``
numeri: i numeri non hanno nulla oltre loro stessi per esempio var x = 1;
array:gli array sono introdotti da delle parentesi quadre e ogni stringa viene divisa da una virgola tipo let array =[a,b,c]
oggetti:gli oggetti sono introdotti da delle parentesi graffe e ogni propietà è divisa da una virgola e scritta tipo property:any,
dove property rappresenta il nome della propietà, i due punti l'inizio della propietà e la virgola la divide dalla prossima.
